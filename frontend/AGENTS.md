# Frontend Agent Notes

Follow the root `AGENTS.md` first.

This frontend uses Next.js, React, TypeScript, and Tailwind CSS. Keep components
simple, conventional, and readable. Use the standard `app/`, `components/`,
`lib/`, and `types/` structure as the project grows.

Do not put authoritative accounting calculations in browser-only code. The
backend owns official HT, TVA, TTC, and VAT summary calculations.
